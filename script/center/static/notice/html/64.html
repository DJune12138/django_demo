1. 太空尋寶，資源豐厚

活動內容：聯軍派出大量艦隊剿滅了長期盤踞在達拉斯行星帶的太空海盜，激烈的戰鬥摧毀了無數的戰艦和據點，大量的寶藏和戰利品散落行星帶廣袤的空間中。尋寶熱潮，傾情來襲，前往該行星帶，和熱情的星際民眾一起，探尋海量的金幣和戰艦裝備！
活動時間：永久

活動要求：玩家達到30級后即可參與
活動入口：從主星界面“太空尋寶”按鈕進入


2. 簽到豪禮，登錄即送

活動內容：多重好禮慶開服，每天登錄遊戲領豪禮，豐厚金幣、鑚石領不停，第2天登入還可以獲得A級推圖英雄，登入達７天稀有紅裝立即贈，登入達15和28天更有高級武器進階圖紙，助你征戰宇宙。

領取方式：【福利】→【簽到獎勵】



3. 線上禮包，支援不斷

活動內容：漫漫征途，線上好禮來相助。每日只要保持登錄遊戲，每隔一段時間都能獲得系統贈送的豐厚物資，助你提升戰力，了卻後顧之憂。

領取方式：【福利】→【線上禮包】


4. 等級成長，禮包相伴

活動時間：永久

活動內容：從10級開始，指揮官等級每提升10級，即可在商城中購買相應的超值等級禮包，每個等級禮包只能購買一次。

領取方式：在【商城】內購買
